ngx-feedback
What is ngx-feeback?

An Angular library helping devs to easily collect Wep App users reviews and real time feedbacks about the product, new features or potential bugs.

Ngx-feedback relies upon html2canvas library to take webpages screenshots.

Features
  • A feedback button you can place at the edge of the page.
  • A screenshot of the whole page or of a particular feature/element.
  • A modal form in which the user could add his comment about the product and vote thanks to a star rating component.
Install

npm i ngx-feedback

or

yarn add ngx-feedback
